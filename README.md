# Tiger Estate Price

Initial project for testing the MOLIT apartment trade API and later building a Seoul apartment record-high price website.

## Setup

1. Install packages.

```bash
pip install -r requirements.txt
```

2. Create a `.env` file by copying the shape of `.env.example`.

```bash
MOLIT_API_KEY=your_molit_api_key_here
```

3. Run the API test.

```bash
python main.py
```

Do not upload `.env` to GitHub because it contains your API key.
